# Assets Folder

Place images and other media files for this presentation here.

## Usage

Reference assets in your slides using relative paths:

```html
<img src="assets/my-image.png" alt="Description">
```

## Recommended Formats

- **Images:** PNG or JPG (PNG for screenshots/diagrams, JPG for photos)
- **Max dimensions:** 1920x1080 (matches slide dimensions)
- **File naming:** Use lowercase with hyphens (e.g., `team-photo.png`)

## Tips

- Keep images as small as possible while maintaining quality
- Use descriptive filenames
- Add alt text for accessibility
