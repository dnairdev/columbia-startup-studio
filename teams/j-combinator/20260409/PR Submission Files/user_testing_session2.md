# User Testing — Round 2
**Session 2**

---

**Participant:** Tiffany Tsai

---

## Key Observations

- Could not find the sign-up button — the entry point for new users was not visible enough.
- Was not informed that a confirmation email would be sent after sign-up, causing confusion and drop-off.
- Did not sign back in after completing sign-up — the post-registration flow lacked clear guidance.
- Did not understand why only three event options were shown — the limited display felt incomplete.
- App surfaced events that had already expired, undermining trust in recommendation quality.
- Did not notice the left-swipe interaction for disliking events — the gesture is not discoverable.
- Hoped the app could support automatic booking or ticket purchasing, similar to Resy or Trip.com.
