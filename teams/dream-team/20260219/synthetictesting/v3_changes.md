# Changes from V2

- **Section:** Hero headline
  **Change:** Replaced "Stop Emailing Into the Void" with "Recruiting, simplified." Shifted from pain-led to benefit-led framing.
  **Why:** The pain headline tested well on resonance but younger personas (HS athletes) said the page felt like a SaaS product, not something aimed at them. Testing a shorter, cleaner headline that feels less corporate and more like a product tagline.

- **Section:** Hero subheadline
  **Change:** Replaced product description with an emotional framing: "A sports recruiting platform that feels like social media — because that's where the talent already is."
  **Why:** 8 personas flagged the tone as too corporate for the target age group. Leaning into the social-media-native angle from the brand position to speak the audience's language.

- **Section:** Value props
  **Change:** Replaced the two-block athlete/coach breakdown with two short taglines: a "core idea" line (LinkedIn meets TikTok) and a "coach value" line (video-first profiles coaches can evaluate in seconds).
  **Why:** 15 personas said the page reads like a SaaS landing page. Compressing the value prop into punchy one-liners instead of paragraphs. Testing whether less structure converts better with younger audiences.

- **Section:** Product demo
  **Change:** Added a video/demo placeholder. The page now features an embedded product demo video showing the feed experience.
  **Why:** 11 personas asked to see the product rather than read about it. This was the single most requested element in V2 feedback. A visual demo addresses the "show me" objection directly.

- **Section:** Differentiation block
  **Change:** Removed the explicit NCSA/Hudl comparison section.
  **Why:** The differentiation objection dropped from 19 mentions (V1) to 6 (V2). It's no longer the top concern. The demo video now does the differentiation work visually — showing a product that clearly isn't NCSA or Hudl.

- **Section:** Trust block
  **Change:** Removed the standalone trust section. Verified coach messaging is now implied through "Coach Mode" UI element rather than stated explicitly.
  **Why:** "No fake accounts, no spam" still tested well but the section felt redundant in a shorter page. Moving trust signals into the product UI itself rather than copy.

- **Section:** Overall structure
  **Change:** Moved from a 5-section layout (hero, value props, differentiation, trust, CTA) to a 3-section layout (hero with taglines, demo video, email capture).
  **Why:** Responding to "feels like a SaaS page" feedback. Testing a minimal, scroll-light layout that feels more like a product launch page than a B2B landing page.
